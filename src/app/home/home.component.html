ASAG dataset editor

This is an editor for the Texas dataset for the Automated Short Answer Grading task which was built from several tests in the computer science domain in english.

The content in spanish was generated by performing an automatic translation to spanish using the Cloud Translation service provided by Google Cloud and then checked by a group of human translators.

Within this editor software you may perform the following actions:

  • Check or update the translations.
  • View details about the questions.
  • View details about the student-answers.
